# w-uan

## 使用

- 获取 url 参数

uan.getUrlParams(参数)
uan.getUrlParams() 所有

- 去除空格

uan.trim(' 123  3 ',1);

- 大小写转换

uan.changeLetter('ahdngfl',1)

- 计数数组中值的出现次数

uan.countOccurrences([1,2,3,4,1,2,3,1],1)

- 设置 cookie

uan.setCookie('COOKIE', 'testcookie', 1)

- 获取 cookie

uan.getCookie('COOKIE')

- 删除 cookie

uan.changeLetter('COOKIE')
